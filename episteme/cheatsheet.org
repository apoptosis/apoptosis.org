#+title:Epi Cheatsheet

*Escape Hatches*
| =C-g= | abort prompt  |
| =q=   | exit debugger |

*Characters*
| =C-f= / =C-b=   | left / right |
| =C-d=         | delete right |
| =<Backspace>= | delete left  |

*Words*
| =M-f= / =M-b=     | left / right |
| =M-d=           | delete right |
| =M-<Backspace>= | delete left  |

*Lines*
| =C-n= / =C-p= | up / down          |
| =C-a= / =C-e= | start / end        |
| =M-<= / =M->= | top / bottom       |
| =C-k=       | cut to end-of-line |

*Copy Paste*
| =C-<Space>= | start highlight |
| =C-w=       | cut highlight   |
| =M-w=       | copy highlight  |
| =C-y=       | paste           |
| =C-g=       | abort           |

*Buffers*
| =C-x b= | switch |
| =C-x k= | kill   |

*Menus*
| =C-c x=     | open menu           |
| =C-c x o=   | open knowledge-base |
| =C-c x U=   | open user-guide     |
| =C-c x p p= | update              |
